# Monster Hunter Wilds Charm Generator

A simple **charm generator** for *Monster Hunter Wilds*, built with **Python + Tkinter**.  
Generates random charm stats following in-game probability patterns.

## How to run
Install the .exe file from **[Releases](https://github.com/MaximilianLJ/Wilds-Charm-Gen/releases/tag/v1.0)**
